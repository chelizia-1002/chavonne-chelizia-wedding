# Chavonne & Chelizia Wedding Website

Static GitHub Pages website.

## Edit RSVP email
Open `script.js` and replace `biancalotter@gmail.com` with the email address that must receive RSVP messages.

## Add your own moving videos
Place `.mp4` files in `assets/videos/` and replace the CSS animated backgrounds with video tags if desired. The current version uses moving cinematic CSS nature layers so it works immediately on GitHub Pages.
